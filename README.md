# Clover's Tools

个人小工具合集。

## 工具列表

| 工具 | 说明 | 技术栈 |
|------|------|--------|
| [Figma 3Dtool](figma-3dtool/) | Figma 插件：对选中元素施加 3D 透视旋转并导出透明 PNG | Figma Plugin API |
| [CutImg](cutimg/) | 桌面应用：按网格批量切分大图，拖拽导入 + 实时预览 | Tauri 2 / Vue 3 / Rust |
| [ReIcon](reicon/) | 桌面工具：PNG 转 ICO 并一键应用到文件夹/快捷方式 | Python / tkinter |
| [FitGuard](fitguard/) | Web 工具：游戏背景图多分辨率预览 (4:3/16:9/21:9/32:9) | HTML / Canvas |
| [RenameTools](renametools/) | 桌面工具：图片批量重命名，前缀 + 自定义文本 + 自动编号 | Python / tkinter |
