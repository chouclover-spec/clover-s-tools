# 3D 变换 - Figma 插件

在 Figma 中实现类似 Adobe XD（Ctrl+T）的 3D 透视变换功能。选中任意元素，通过三轴旋转参数实时预览 3D 效果，一键生成带透明背景的 PNG 图片放置在画布上。

## 功能

- **三轴旋转**：独立控制 X / Y / Z 轴旋转角度
- **实时预览**：拖动滑块即时在插件面板内看到 3D 效果
- **透明背景**：生成的 PNG 保留 Alpha 通道，可叠加在其他元素上
- **高清输出**：以 2x 分辨率渲染，保证图片清晰度
- **非破坏性**：原始元素完整保留，结果作为新图片节点放置在右侧
- **可重复编辑**：选中已生成的 3D 图片再次运行插件，自动溯源到原始元素，角度从 0° 重新开始
- **主题适配**：自动跟随 Figma 亮色 / 暗色主题

## 使用方法

1. 在 Figma 画布上**选中一个元素**（矩形、文字、组、Frame、组件等均支持）
2. 运行插件 **3D 变换**
3. 在插件面板内调整旋转角度：
   - **X 轴旋转**：-90° ～ 90°（上下翻转）
   - **Y 轴旋转**：-90° ～ 90°（左右翻转）
   - **Z 轴旋转**：-180° ～ 180°（平面旋转）
4. 点击 **应用**，3D 效果图片将出现在原元素右侧
5. 点击 **重置** 可将所有角度归零

**再次编辑：** 选中已生成的 `[3D变换]` 图片节点，重新运行插件，插件会自动找回原始元素并以 0° 为起点重新编辑，应用后原地更新图片。

## 安装（本地开发）

1. 克隆或下载本仓库到本地

2. 打开 **Figma Desktop**（网页版不支持本地插件开发）

3. 菜单栏：**Plugins → Development → Import plugin from manifest...**

4. 选择项目目录下的 `manifest.json`

5. 在画布中选中元素后，通过以下方式运行插件：
   - 菜单：Plugins → Development → 3D 变换
   - 或使用快捷键 `Ctrl+/`（Mac: `⌘/`）搜索插件名称

## 文件结构

```
3Dtool/
├── manifest.json   # 插件清单（名称、入口、权限配置）
├── code.js         # 主线程：选区校验、PNG 导出、结果节点创建
└── ui.html         # 插件 UI：WebGL 实时预览 + 控件 + 应用逻辑
```

无需 Node.js、npm 或任何构建工具，直接导入即可使用。

## 技术实现

| 层 | 技术 |
|---|---|
| 插件主线程 | Figma Plugin API（`exportAsync`、`createImage`、`createRectangle`） |
| UI 渲染 | 原生 WebGL（无外部依赖） |
| 3D 变换 | 手写 4×4 矩阵数学（旋转矩阵 + 透视投影） |
| 图片导出 | `canvas.toBlob()` → `Uint8Array` → Figma `IMAGE` fill |

**数据流：**

```
选中元素
  → exportAsync(PNG, 2x)
  → WebGL 纹理渲染（实时预览）
  → 用户调整 X/Y/Z 旋转
  → 点击"应用"
  → canvas.toBlob → PNG 字节
  → figma.createImage → Rectangle [3D变换]
  → 放置在原元素右侧 50px
```

## 注意事项

- 输出图片最大分辨率为 **4096 × 4096**（Figma 图片限制），超出时自动缩放
- X / Y 轴接近 ±90° 时图形趋于消失（侧面透视），属正常效果
- 首次发布到 Figma 社区时，需在 Figma Desktop 中通过 **Create new Plugin** 获取真实插件 ID，并更新 `manifest.json` 中的 `id` 字段
