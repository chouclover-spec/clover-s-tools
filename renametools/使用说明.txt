===============================================
        图片批量重命名工具 - 使用说明
===============================================

【工具简介】
这是一个简单易用的图片批量重命名工具，可以帮助您快速整理和重命名大量图片文件。
支持PNG和JPG格式的图片。


【快速开始】

第一步：启动程序
-----------------
双击运行 "start.bat" 文件即可启动程序。
（首次运行可能需要几秒钟时间加载）


第二步：选择图片
-----------------
1. 点击窗口中的"选择文件夹"按钮
2. 在弹出的对话框中，选择一个或多个需要重命名的图片
   - 按住 Ctrl 键可以多选图片
   - 点击"全选"按钮可以选择所有图片
3. 点击"确定"确认选择


第三步：设置重命名规则
-----------------------
重命名格式：A_B_数字编号.扩展名

▶ 前缀A（下拉选择）：
  - （空）    - 不添加前缀
  - Bg        - 背景图片
  - Icon      - 图标
  - Img       - 图片
  - Btn       - 按钮
  - Bar       - 栏/条

▶ 中间B（自己输入）：
  - 可以输入任何文字描述
  - 也可以留空不填

▶ 数字编号（自动生成）：
  - 如果选择的图片数量 ≤ 100张：从 01 开始（01, 02, 03...）
  - 如果选择的图片数量 > 100张：从 001 开始（001, 002, 003...）
========================================
图片重命名工具 - 使用说明
========================================

【软件介绍】

这是一个简单易用的图片批量重命名工具，无需任何代码基础即可使用。

功能特点：
✓ 可视化界面，操作简单
✓ 支持 PNG 和 JPG 格式图片
✓ 支持单张或批量重命名
✓ 实时图片预览
✓ 智能编号，避免文件名冲突

========================================
【使用步骤】
========================================

第一步：启动程序
-----------------
双击运行"图片重命名工具.exe"

第二步：选择文件夹
-----------------
1. 点击界面左上角的"选择文件夹"按钮
2. 找到存放图片的文件夹
3. 点击"选择文件夹"确认

第三步：选择要重命名的图片
-----------------
在左侧图片列表中：
- 单击图片名称：选中一张图片（右侧会显示预览）
- 按住 Ctrl + 单击：可以选中多张图片

第四步：设置新文件名
-----------------
在右侧面板中输入三个部分：

┌─────────────────────────┐
│ A值：产品              │  ← 第一部分
│ B值：红色              │  ← 第二部分  
│ C值：正面              │  ← 第三部分
└─────────────────────────┘

最终文件名格式：A_B_C_编号.png
例如：产品_红色_正面_001.png

第五步：执行重命名
-----------------
- 重命名一张：点击"重命名选中图片"
- 批量重命名：选中多张后点击"批量重命名"

完成！

========================================
【重要提示】
========================================

✓ 编号自动递增
  如果文件夹中已有"产品_红色_正面_001.png"
  新文件会自动从 002 开始编号

✓ 重命名前请确认
  重命名后原文件名会被替换，无法撤销
  重要文件建议先备份

✓ 支持的格式
  目前支持：.png、.jpg、.jpeg
  其他格式暂不支持

========================================
【常见问题】
========================================

Q1：双击程序后没有反应？
A1：可能被杀毒软件拦截了
    - 检查杀毒软件提示
    - 将程序添加到信任列表

Q2：看不到图片预览？
A2：可能是图片文件损坏或格式不支持
    - 确认图片是 PNG 或 JPG 格式
    - 尝试用其他软件打开图片测试

Q3：重命名后文件找不到了？
A3：文件还在原文件夹，只是名字变了
    - 在原文件夹中按修改时间排序查找

Q4：可以撤销重命名吗？
A4：目前不支持撤销功能
    - 建议重要文件先复制一份备份

Q5：能同时处理多个文件夹吗？
A5：每次只能处理一个文件夹
    - 需要处理多个文件夹请分别操作

【使用示例】
========================================

场景：给商品图片批量重命名

原文件名：
  IMG_001.jpg
  IMG_002.jpg
  IMG_003.jpg

操作：
  1. 选择这三张图片
  2. 输入 A值：手机壳
  3. 输入 B值：透明
  4. 输入 C值：展示
  5. 点击"批量重命名"

新文件名：
  手机壳_透明_展示_001.jpg
  手机壳_透明_展示_002.jpg
  手机壳_透明_展示_003.jpg
