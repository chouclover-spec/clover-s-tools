# 图片重命名工具

一个简单易用的图片重命名工具，支持批量重命名PNG和JPG格式的图片文件。

## 功能特点

- 📁 选择文件夹并浏览其中的图片
- 🖼️ 单选或多选需要重命名的图片
- 🎨 支持PNG和JPG格式
- 📝 灵活的重命名规则：A_B_C格式
- 👀 图片预览功能（异步加载，不阻塞界面）
- 🔢 智能编号：自动检测已存在的文件名，避免冲突

## 安装步骤

### 1. 创建虚拟环境

在项目目录下打开命令行，执行：

```bash
# Windows
python -m venv venv
venv\Scripts\activate

# Linux/Mac
python3 -m venv venv
source venv/bin/activate
```

### 2. 安装依赖

```bash
pip install -r requirements.txt
```

## 使用方法

### 1. 启动程序

```bash
python image_renamer.py
```

### 2. 操作步骤

1. **选择文件夹**：点击"选择文件夹"按钮，选择包含图片的文件夹
2. **选择图片**：在弹出的对话框中选择需要重命名的图片（支持多选）
   - 可以点击"全选"快速选择所有图片
   - 点击"确定"确认选择
3. **设置重命名规则**：
   - **前缀A**：下拉选择，可选值：
     - （空）- 不添加前缀
     - Bg - 背景图片
     - Icon - 图标
     - Img - 图片
     - Btn - 按钮
     - Bar - 栏/条
   - **中间B**：用户自定义文本（可选，留空则不添加）
4. **预览图片**：在左侧列表中选择图片，右侧会显示预览
5. **执行重命名**：点击"执行重命名"按钮，确认后执行

### 3. 重命名规则说明

重命名格式：`A_B_C.扩展名`

- **A**：前缀选项（可选）
- **B**：用户自定义文本（可选）
- **C**：数字编号
  - 如果选择图片数量 ≤ 100：从 `01` 开始递增
  - 如果选择图片数量 > 100：从 `001` 开始递增
  - 如果文件夹中已存在相同格式的文件名，会自动从下一个可用编号开始

**示例**：
- 选择前缀A="Icon"，中间B="Button"，3张图片
  - 结果：`Icon_Button_01.png`, `Icon_Button_02.png`, `Icon_Button_03.png`
- 选择前缀A="Bg"，中间B为空，2张图片，且文件夹中已有`Bg_01.jpg`
  - 结果：`Bg_02.jpg`, `Bg_03.jpg`

## 注意事项

- 重命名操作不可撤销，请谨慎操作
- 如果目标文件名已存在，会跳过该文件并提示
- 图片预览采用异步加载，不会阻塞界面操作
- 支持的图片格式：PNG、JPG、JPEG（不区分大小写）

## 系统要求

- Python 3.7+
- Windows / Linux / macOS

## 依赖库

- Pillow >= 10.0.0（用于图片处理和预览）
