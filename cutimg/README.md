# CutImg — 图片批量裁切工具

> 专为美术/设计同学打造的轻量桌面工具，将一张大图按指定行列数切成若干小块，批量导出。

## 功能说明

| 功能 | 说明 |
|------|------|
| **批量导入** | 支持一次选择多张图片，也支持直接拖拽图片到窗口 |
| **等分裁切** | 按设定的"列数 × 行数"将图片均等分割 |
| **实时预览** | 中间预览区会显示红色分割线和块编号，所见即所得 |
| **批量导出** | 支持对所有图片使用相同参数一键全部导出 |
| **路径记忆** | 自动记住上次使用的导出文件夹，下次无需重选 |
| **格式保留** | 导出文件与源文件保持相同格式（PNG / JPG / WebP / GIF） |

## 使用步骤

### 第一步：导入图片
- **方式 A**：点击左侧「点击选择 或 拖拽图片到此处」区域，在弹出窗口中选择图片
- **方式 B**：直接将图片文件从文件夹**拖入**程序窗口

支持格式：`PNG`、`JPG/JPEG`、`WebP`、`GIF`

---

### 第二步：设置裁切参数
在右侧控制面板中调整：

| 参数 | 说明 | 示例 |
|------|------|------|
| **横向列数（X）** | 横向切几列 | 设为 `3` = 左中右三列 |
| **纵向行数（Y）** | 纵向切几行 | 设为 `3` = 上中下三行 |

> 例：设置 **3列 × 3行** → 一张图切成 **9 块**，按从左到右、从上到下编号 1~9。

中间预览区会**实时显示**分割线和块编号，确认无误后再导出。

---

### 第三步：选择导出路径
点击「导出路径」框，选择想要保存裁切结果的文件夹。  
程序会自动记住你上次选择的路径，下次打开无需重选。

---

### 第四步：导出
- **只导出当前图片**：取消勾选「对所有图片使用相同参数」，点击「开始裁切导出」
- **批量导出全部图片**：勾选「对所有图片使用相同参数」，按钮会显示「导出全部 N 张」

**导出文件命名规则：**  
```
原文件名_编号.扩展名
```
例：源文件为 `sprite.png`，3×3 裁切后生成：  
`sprite_1.png` `sprite_2.png` … `sprite_9.png`

## 常见问题

**Q：图片导入后没有显示宽高？**  
A：稍等片刻，程序正在读取图片元数据，大图可能需要 1~2 秒。

**Q：裁切后图片尺寸不均等？**  
A：当原始图片宽/高不能被列/行数整除时，最后一列/行会稍大一点（补足剩余像素），这是正常行为。

**Q：支持 PSD / AI 格式吗？**  
A：暂不支持，请先在 PS 或 AI 中导出为 PNG/JPG 再使用本工具。
